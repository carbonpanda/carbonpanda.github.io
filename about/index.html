This site

I will use this site to record my home automation setup. It will serve as my build log so I can refer to it later for any enhancement. There is always room for improvement.

If you are looking for some sort of reference projects on your own home automation development, I do hope you will find my materials helpful.
Currently my setup only consists of a Raspberry Pi 3B running Home Assistant Hass.io with Mosquitto MQTT server. A Sonoff Basic on Tasmota controlling a water heater. It is interfaced to Siri and Google Assistant.
